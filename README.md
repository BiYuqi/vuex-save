## [vuex-save](https://github.com/BiYuqi/vuex-save)

> 自动本地化存储vuex数据, 解决刷新页面vuex数据丢失

## 安装
```js
npm install vuex-save
```
## 使用
```js
import vuexSave from 'vuex-save'

// store.js
const store = new Vuex.Store({
  state,
  mutations,
  plugins: [myPlugin({
    // set save mode
    mode: 'localStorage'
  })]
})
```
